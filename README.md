# 🕹️ Neon Tic-Tac-Toe | Premium Edition

A modern, high-fidelity Tic-Tac-Toe game built with a premium **Neon Glassmorphism** aesthetic. Experience the classic game with vibrant visuals, smooth animations, and a sleek dark-mode interface.

![Tic-Tac-Toe Preview](https://raw.githubusercontent.com/adityayadav100c-spec/tic-tac-toe-game/main/screenshot.png) *(Note: Add your actual screenshot here)*

## ✨ Features

- **Premium Design**: Dark mode base with neon Cyan (X) and Neon Pink (O) accents.
- **Glassmorphism UI**: Uses modern CSS `backdrop-filter` for a frosted glass effect.
- **Dynamic Animations**: Smooth transitions for placing marks and pulsing winning lines.
- **Local Multiplayer**: Two-player turn-based gameplay.
- **Score Tracking**: Persistent score tracking for the current session.
- **Fully Responsive**: Optimized for Mobile, Tablet, and Desktop screens.
- **Fast Development**: Powered by [Vite](https://vitejs.dev/) for an ultra-fast developer experience.

## 🚀 Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Vanilla CSS with custom properties (variables), Flexbox, Grid, and Glassmorphism.
- **JavaScript**: Clean, modular DOM-based game logic.
- **Vite**: Modern frontend tooling.

## 🛠️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v14.0.0 or higher)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/adityayadav100c-spec/tic-tac-toe-game.git
   ```

2. Navigate to the project directory:
   ```bash
   cd tic-tac-toe-game
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to `http://localhost:5173`.

## 🎮 How to Play

- Player X always starts first.
- Players take turns clicking on empty cells to place their marks.
- The first player to get 3 of their marks in a horizontal, vertical, or diagonal row wins!
- If all 9 cells are filled without a winner, the game is a draw.
- Use the **Restart Game** button to reset the board while keeping the scores.
- Use **Clear Scores** to reset everything to zero.
